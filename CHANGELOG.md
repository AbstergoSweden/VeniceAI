# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-08

### Added

- Complete project release.
- Uncensored AI image generation with multiple models.
- Integrated chat interface with memory and vision support.
- Persistent history using Firebase (optional).
- Web3 transaction support with MetaMask.
- Electron desktop application wrapper.
- Automatic model synchronization script (`scripts/venice_model_sync.py`).
- Comprehensive provider configuration (`providers.yaml`).
